Job Application Tracker

A web-based job application tracker built with Python and Flask. Track every company you apply to, update application status, and monitor your job search progress from a clean browser interface.

Features
- Add job applications with company and role
- View all applications in a formatted table
- Update status - Applied, Interview, Offer, Rejected
- Delete applications
- Dashboard showing total, interviews, offers, and rejections

Tech Stack
- Python 3
- Flask
- SQLAlchemy
- SQLite
- HTML, CSS, Jinja2

How to Run
git clone https://github.com/SonuSRaj123/job-tracker.git
cd job-tracker
pip install flask flask-sqlalchemy
python app.py

Then open http://127.0.0.1:5000 in your browser.
